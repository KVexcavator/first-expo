•Rendering data collections

Компонент React Native для рендеринга списков, — FlatList, который работает одинаково на iOS и Android. Представления списков принимают свойство data, которое является массивом объектов. Эти объекты могут иметь любые свойства, но им требуется свойство key. Если нет свойства key, необжодимо передать свойство keyExtractor компоненту Flatlist и указать, что использовать вместо key.

•Sorting and filtering lists

• ListContainer: общий контейнер для списка; он следует знакомому шаблону контейнера React
• List: компонент без состояния, который передает соответствующие части состояния в ListControls и компонент React Native ListView
• ListControls: компонент, который содержит различные элементы управления, изменяющие состояние списка
• ListFilter: элемент управления для фильтрации списка элементов
• ListSort: элемент управления для изменения порядка сортировки списка
• FlatList: фактический компонент React Native, который отображает элементы
