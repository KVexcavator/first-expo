==========================================
Expo CLI — это часть большой экосистемы для разработки приложений React Native. Expo — это фреймворк и платформа для универсальных приложений React. Созданный на основе React Native и собственных платформ, он позволяет вам создавать, развертывать, тестировать и быстро итерировать приложения iOS, Android и веб-приложения из единой кодовой базы JavaScript/TypeScript.

Фреймворк Expo предоставляет следующее:
• Expo CLI: инструмент командной строки, который может создавать пустые проекты, а затем запускать, собирать и обновлять их.
• Expo Go: приложение для Android и iOS для запуска ваших проектов непосредственно на вашем устройстве (без необходимости компилировать и подписывать собственные приложения) и делиться ими со всей вашей командой.
• Expo Snack: онлайн-площадка, которая позволяет вам разрабатывать приложения React Native в браузере.
• Expo Application Services (EAS): набор глубоко интегрированных облачных сервисов для приложений Expo и React Native. Приложения можно компилировать, подписывать и загружать в магазины с помощью EAS в облаке.

установка и использование инструмента командной строки Expo
https://docs.expo.dev/get-started/installation/#requirements
Необходимо установить Node.js, Git и Watchman.
-----------------------------------------------------------------
Watchman — это инструмент для наблюдения за файлами в нашем проекте, чтобы запускать такие действия, как пересборка, при их изменении.
https://facebook.github.io/watchman/docs/install
Building from Source
https://ubuntu.pkgs.org/22.04/ubuntu-universe-arm64/watchman_4.9.0-6build2_arm64.deb.html
$ sudo apt install watchman
$ watchman version
$ man watchman
npx create-expo-app my-app
cd my-app
npx expo start
Проверка работы Watchman
watchman watch-list
